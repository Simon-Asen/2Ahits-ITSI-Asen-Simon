## grep
Mit dem Befehl grep kann man nach bestimmten Wörtern in einer Datei suchen:
```sh
grep Mia klassenkassa.csv
```
Ausgabe:
```sh
Mia,29.09.2024,2.11,Kino
Mia,29.09.2024,2.11,Kino
Mia,07.11.2024,19.45,Walter
Mia,13.10.2024,10.56,Kopierkosten
Mia,09.11.2024,11.45,Kino
```
Überall wo Mia in der Datei klassenkassa.csv vorkommt,
wird es Ausgegeben.

## Piepes
Sie können die Ausgabe eines Befehls in die Eingabe eines anderen lenken.

Eingabe:
```sh
head klassenkassa.csv | grep Werkstatt
```
Ausgabe:
```sh
Sophia,22.09.2023,15.0,Werkstatt
Lena,18.09.2023,13.0,Werkstatt
Emma,22.09.2023,42.0,Werkstatt
Nico,18.09.2023,23.0,Werkstatt
```
