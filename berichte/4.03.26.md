## Übung 2
Mit der Datei shopping.txt – löse jeden Punkt durch einen Einzeiler mit Hilfe von Pipes:
- Sortiere in umgekehrter Reihenfolge (eine Option von sort)
- Sortiere nach der 2ten Spalte – Obstnamen (dies ist ebenfalls eine Option von sort)
- Nimm alles bis auf die ersten 5 und letzten 5 Zeilen, sortiere das nach Personennamen in absteigender Reihenfolge ('Z' zuerst). Sortiere das Ergebnis aufsteigend nach Obstnamen und gibt die letzten 3 Zeilen des Endergebnisses aus.

```sh
cat shopping.txt | sort -r #von Z nach A sortiert
```
```sh
cat shopping.txt | sort -t ' ' -k2 # sortiert nach der 2-ten Spalte
```
```sh
tail -n +6 shopping.txt | head -n5 | sort -r | sort -t ' ' -k2 | tail -n3 #💀
```
## Übung 3
Erstelle mit cut, sort und uniq eine alphabetisch sortierte Liste an Namen die in shopping.txt vorkommen. Jeder Name soll in dieser Liste nur einmal stehen.
```sh
cut -d' ' -f1 shopping.txt | sort | uniq # cut -d ' ' wird jede zeile im den fall in 3 felder geteilt. -f1 erstes feld wird ausgeschnitten
```
## Übung 4
Gib von shopping.txt alles bis auf die ersten zwei und die letzten zwei Zeilen aus. Hinweis: die -n Option von head und tail erlaubt dies durch eine spezielle Angabe bei der Zeilenanzahl. Löse durch einen Einzeiler mit Hilfe von Pipes.

```sh
tail -n +3 shopping.txt | head -n -2 # -n +3 beginnt erst ab zeile 3 zu zählen, -n -2 letzten 2 zeilen auslassen
```
