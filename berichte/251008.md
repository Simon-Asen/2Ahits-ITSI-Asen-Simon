## Grundlagen

### Optionen
  - shorthand `-l` (kurze Schreibweise)
  - longhand `--long` (lange Schreinweise)

### Argumente
- `echo` (texte und variablen ausgeben)
- Histoy (schpeichert alle eingegebenen Befehle)
### Tastekürzel
- `strg + C` Befehl abbrechen
- `strg + L` = `clear` alle Befehle aus der Anzeige weg
  

### Filesystem
  - `ls` Inhalt anzeigen
  - `cd` Arbeitsverzeichnis wechseln
  - `pwd` zeigt aktuellen Pfad an
  - `mkdir` neuse Verzeichnis
  - `rm` Datei löschen
  - `rmdir` leere Verzeichnise entfernen
  

absoluter Pfad: immer mit `/`, genauer Ort einer Datei,

relativer Pfad: Weg vom aktuellen Verzeichnis ausgehend,


  
`rm -r /` löscht vom Wurzelverzeichnis ausgehend alles

`ls /` zeigt den Inhalt des Wurzelverzeichnisses an

### Beispiel:
```sh
bin   dev  home    io   lib64  mnt  opt   proc  root  sbin  store  tmp  var
boot  etc  inject  lib  media  nix  pid2  repl  run   srv   sys    usr
```
