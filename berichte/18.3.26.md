## passwort Check manipulieren
datei herunterladen mit:
```sh
curl -O https://www.franzmatejka.at/htl/doc/ITSI_2_linux/opcode_patching/bin/replit/pwcheck
```

Ausführbar machen:
```sh
chmod u+x pwcheck
```

Binärcode ansehen:
```sh
xxd pwcheck
```

GNU Debugger Installieren(gdb);
```sh
gdbpwcheck
```

Programm mit dem Debugger öffnen:
```sh
gdb pwcheck
```
Breakpoint im Main setzen:
```sh
b main
```
Starten:
```sh
run
```
Opcode ansehen:
```sh
disas Main
```
 ### Passwort manipulieren:
- nach einen compare Befehl suchen, der der die Eingabe mit dem Original vergleicht.
- danach kommt ein Jump Befehl. Den mit einen `nop` Befehl austauschen:
  ```sh
  set {unsigned char}Speicheradresse des jump Befehls = 0x90
  ```
- für alle Bytes des jump-Befehls wiederholen
- mit `c` das Programm weiterführen
- das Passwort wird abgefragt und danach angezeigt, auch wenn man es falsch eingegeben hat.
